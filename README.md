# Community Event Finder (Backend)

## Tech Stack
- Node.js
- Express
- TypeScript
- MongoDB
- JWT Authentication
- Zod Validation
- Nodemailer

## Features
- User registration with OTP email verification
- Login with JWT
- Role-based access
- Create / Join Events
- Location filtering
- Dashboard analytics
- Popular events aggregation

## Setup

1. Clone repo
2. Install dependencies
3. Add .env file
4. Run server
